Привет уважаемый читатель!

При написании этого проекта использовался tomcat 7 и xampp для работы с mysql.

Перед началом работы необходимо развернуть бекап базы данных из папки Database.

Проверить наличие системной переменной Catalina_home.

В файле build.xml изменить поля login и password на соот. пользователю из файла tomcat-users.xml
